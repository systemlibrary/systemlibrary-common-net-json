    PartialJson<T>(String, String, JsonSerializerOptions)

    Search through the json to find a property, and returns its value as T

    Supports a path of all properties seperated by a slash to the leaf-property that will be translated into the T

    Throw exception if none of the parent properties exists

    Search is case-insensitive

    Returns null if the leaf property do not exist in the json data

    Declaration
    public static T PartialJson<T>(this string json, string findPropertySearchPath = null, JsonSerializerOptions options = null)
    where T : class
    Parameters
    Type Name Description
    System.String json

    Json formatted string
    System.String findPropertySearchPath

    Name of the property that will be deserialized as T

    Supports search path to the leaf-property, such as: parent1/parent2/leaf where leaf will be deserialized as T

    NOTE: If T Example: deactivated/users

    Will search for property "deactivated", if found, it will search for a "users" property within that object If "deactivated", or any other part of the path is not found, it will throw exception

    If the leaf property, in our case "users" is not found, it will return null
    System.Text.Json.JsonSerializerOptions options
    Returns
    Type Description
    T

    Returns json as T or null if not found
    Type Parameters
    Name Description
    T

    A class or a List/Array of a class

    NOTE: If T is a List/Array and no findPropertySearchPath is specified the Searcher auto-adds the suffix sealed. For instance List<user> will look for the property "users"
    Examples

    //Assume json data: { "users" [ ... ] } var res1 = data.PartialJson<List<User>>(); //Searches for a property "users", case insensitive, due to the objet is a List the S is appended

    //Assume json data: { "users" [ ... ], "deactivated": { "users": [ ... ] } } var res2 = data.PartialJson<List<User>>("deactivated/users"); //Searches for a property "users" inside the root property "deactivated"
